# `Java`内存模型`FAQ`（`六`）没有正确同步的含义是什么？
没有正确同步的代码对于不同的人来说可能会有不同的理解。在`Java`内存模型这个语义环境下，我们谈到“没有正确同步”，我们的意思是：

一个线程中有一个对变量的写操作，
另外一个线程对同一个变量有读操作，
而且写操作和读操作没有通过同步来保证顺序。
当这些规则被违反的时候，我们就说在这个变量上有一个“数据竞争”(`data race`)。一个有数据竞争的程序就是一个没有正确同步的程序。
